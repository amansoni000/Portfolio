## 🚀 Setup & Run Locally

Clone the repository:
```bash
git clone https://github.com/amansoni000/Portfolio.git
cd Portfolio
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview production build:
```bash
npm run preview
```
